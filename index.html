The Porphyria is an open source cryptocurrency with a blockchain with the real name of the users on every transaction. Porphyria was created and authored by myself and licensed under the MIT license.

porphyria.ativarsoft.com

Yummy

Yummy is yet another media player. It supports plugins, services, classic skins and modern skins targeting the Winamp API. It does not depend on GTK+, Qt or any other toolkit.

github.com/ativarsoft/yummy

Overkill

Overkill is yet another media player. It does NOT depend on GTK+, Qt or any other toolkit.

Overkill is a rewrite of Yummy in the Ada programming language. This name was chosen because the fact that it is written in Ada makes it secure, fast, and light on memory to an extravagant and hyperbolic point.

github.com/ativarsoft/overkill
